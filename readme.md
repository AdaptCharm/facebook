# Backbone API : Facebook

A simple way to interact with Facebook's Graph API using Backbone.js objects.

## Install 

```
bower install backbone.api.facebook
```
## Dependencies 

* Facebook JS: [all](http://connect.facebook.net/en_US/all.js)
